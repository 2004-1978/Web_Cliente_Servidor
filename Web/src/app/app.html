🌐 ¿Cómo funciona la Web?

Cliente - Servidor - Navegador

👤 Cliente

El cliente es el usuario que solicita información desde internet. Normalmente usamos navegadores como Chrome, Edge o Firefox.

🖥️ Servidor

El servidor almacena páginas web, imágenes, bases de datos y responde a las solicitudes del cliente.

🌍 Navegador

El navegador interpreta HTML, CSS y JavaScript para mostrar correctamente una página web.

🔁 Flujo Cliente - Servidor

1. El usuario escribe una URL.
2. El navegador envía la solicitud.
3. El servidor responde.
4. El navegador muestra la página.

💻 Ejemplo en Código HTML

<h1>Bienvenido a mi página</h1>
<p>Esta página funciona gracias al modelo cliente-servidor</p>
<button>Enviar solicitud</button>

Este código representa una página sencilla. El navegador interpreta estas etiquetas y las muestra visualmente.
